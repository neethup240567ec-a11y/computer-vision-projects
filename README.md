# Computer Vision Projects

This repository contains basic image processing and computer vision projects using **Python & OpenCV**.

##  Projects Included

### 1. Image to Grayscale
- Convert color images to grayscale using OpenCV

### 2. Edge Detection
- Detect edges using Canny edge detection

### 3. Face Detection
- Face detection using Haar Cascade classifier

### 4. Image Enhancement
- Image enhancement using histogram equalization
